Quick Start

The rpRanker workflow is used to predict using retrowynthesis, possible metabolic pathways to produce a compound of interest. To import it, Select this link to import the workflow into your Galaxy instance and select the + on the top right corner of the page:

Import SBML Model

To import SBML models that will work with the workflow, select: Shared Data on the top panel and then select Data Libraries in the scroll down menu.

Thereafter, select Models and import one or more model from the organism of your choosing to your workspace by either clicking on +Datasets and selecting the models to add to a particular history or selecting Export to History and selecting the desired history. Thereafter the models should appear in your history and be available to be used in your workflow/tools runs.



Run the workflow

To run the workflow, select Workflow in the top panel. Then select the Imported: rpRanker workflow dropdown menu and select Run. Three different types of information must be provided:
  • InChI structure of the target molecule you want to produce
  • Maximal length of the predicted pathways
  • A GEM SBML model (as specified above) in which you would like to produce your compound of interest

For example, if you would like to find potential metabolic routes to produce muconic acid in E.Coli, with a maximal pathway of three enzymatic steps, enter the following information in the workflow:
  • InChI=1S/C6H6O4/c7-5(8)3-1-2-4-6(9)10/h1-4H,(H,7,8)(H,9,10)/p-2/b3-1-,4-2-
